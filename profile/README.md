# conrad.gle.bar

Open-source projects from the work of **Conrad Gleber, MD, MBA** — ACMIO & Clinical Informaticist at the University of Rochester Medical Center.

## Repositories

| Repository | Description |
|---|---|
| [1/365 Survey System](https://github.com/conrad-gle-bar/one-threesixtyfive-survey) | A perpetually running survey system that reaches every URMC employee once per year to capture continuous feedback on technology usefulness |
| [Sepsis Retrospective Emails](https://github.com/conrad-gle-bar/sepsis-retrospective-email) | An automated email system for sending personalized retrospective sepsis event data to healthcare providers. Sepsis is defined by the CDC Adult Sepsis Events (ASE) criteria. |
| [Philter Web App](https://github.com/conrad-gle-bar/philter-web-app) | A simple web interface for the Philter PHI (Protected Health Information) filtering tool for clinical notes. |

## About

These projects were designed and built to solve specific problems in a large academic medical center. For more context on the motivation, design decisions, and impact of each project, visit [conrad.gle.bar](http://conrad.gle.bar).
